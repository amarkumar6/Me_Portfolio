# Amar Kumar - Advanced Developer Portfolio

Welcome to my advanced developer portfolio! This is a modern, interactive web-based portfolio showcasing my skills, projects, and professional journey in software development.

## 🌐 Live Demo

**[View Live Portfolio →](https://me-portfolio-vert.vercel.app/)**

## 🚀 Features

- **Responsive Design** - Optimized for all devices and screen sizes
- **Dark/Light Theme Toggle** - Seamless theme switching with persistent storage
- **Color Palettes** - Multiple color scheme options (Teal, Violet, Rose)
- **Recruiter Mode** - Specialized view for hiring professionals
- **Interactive Elements** - Smooth animations, scroll progress tracking, and custom cursor effects
- **Project Showcase** - Detailed project cards with filtering capabilities
- **Skill Visualization** - Interactive skill radar chart
- **Contact Form** - Direct messaging functionality
- **Accessible** - Built with accessibility best practices (ARIA labels, semantic HTML)

## 💻 Tech Stack

### Frontend
- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Grid, Flexbox, animations
- **JavaScript (Vanilla)** - No framework dependencies
- **Google Fonts** - Inter font family

### Skills Showcase
- **Java** - Full stack development
- **MERN Stack** - MongoDB, Express, React, Node.js
- **Python** - Data analysis and machine learning
- **Machine Learning** - AI/ML model development

## 🔧 How to Use

1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd Portfolio
   ```

2. **Open in Browser**
   - Simply open `index.html` in your web browser
   - No build process or dependencies required!

3. **Customize**
   - Update personal information in `index.html`
   - Modify color schemes in `styles.css`
   - Adjust functionality in `script.js`
